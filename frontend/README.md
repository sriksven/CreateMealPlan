# SmartPantry Frontend

A smart kitchen assistant interface built with React, Vite, and TypeScript.

## Architecture

- **Framework**: React (Vite)
- **Language**: TypeScript
- **Styling**: Vanilla CSS (CSS Variables for theming)
- **State Management**: React Hooks
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Directory Structure

- **src/components**: Reusable UI components (Layouts, Buttons, etc.)
- **src/pages**: Route-specific pages (Landing, Login, Dashboard)
- **src/index.css**: Global styles and variables

## Setup and Running

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```
